# UniswapV2Router02

- Bunzz Cliではデプロイできなかったため、Hardhatからデプロイ。

## Clone

```sh
git clone git@github.com:cardene777/base-code.git
```

## Change Directory

```sh
cd Bunzz/contract/UniswapV2/UniswapV2Router02/
```

## Module

```sh
npm install
```

## Compile

```sh
npx hardhat compile
```

## Deploy

```sh
npx hardhat run --network sepolia scripts/deploy.ts
```
