# UniswapV2Router02

- Bunzz Cliではデプロイできなかったため、Hardhatからデプロイ。

## Clone

```sh
npx bunzz clone 1_0xc02aaa39b223fe8d0a0e5c4f27ead9083c756cc2
```

## Compile

```sh
npx hardhat compile
```

## Deploy

```sh
npx hardhat run --network sepolia scripts/deploy.ts
```
